Sobre Mi Proyecto

Este proyecto es basado 100% en instrucciones de SoyHenry, el diseño es de mi autoria y utiliza la libreria de Bootstrap para su diseño, usa JavaScript para su interactividad, y HTML y CSS para su estilizado.
Su backend es basado 100% en instrucciones de SoyHenry y usa la libreria Express para su desarrollo, MongoDB para su base de datos, y Node para su servidor.
El proyecto fue realizado por Lucas Heredia como proyecto integrador del módulo 2
